=========================
Liac Chess (Versão 1.0.0)
=========================

O software **Liac Chess** está sendo desenvolvido pelo `Laboratório de Inteligência Artificial Conexionista (LIAC) <http://inf.ufrgs.br/liac>`_, localizado na Universidade Federal do Rio Grande do Sul (UFRGS). Pensado para ser utilizado nas tarefas didáticas em disciplinas de inteligência artificial, o objetivo do Liac Chess é fornecer uma estrutura gráfica para o jogo de xadrez ou variantes mais simples como a Batalha dos Peões, de forma que os alunos possam se concentrar em desenvolver os algoritmos da disciplina.
